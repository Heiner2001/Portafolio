Proyectos

Una selección de algunos de mis trabajos recientes.

Interfaz web para el control de inventario

Descripción: Este proyecto es una interfaz web en donde se llevaba el control de inventario de una empresa de venta de productos. Técnologías utilizadas: La interfaz fue realizada con C#, Visual Studio Code y se utilizó una base de datos SQL Server. Objetivo principal: El objetivo era tener un mejor control de los productos disponibles que se podian vender y asi llevar un inventario mas eficiente. Resultado obtenido: El resultado obtenido fue un mejor control de los productos disponibles y un mejor manejo de los mismos.

Asistente virtual de apoyo emocional

Descripción: Este proyecto fue la creación de un asistente virtual básico que pudiera ayudar a las personas con ansiedad, depresión y miedo para asi ayudar a la psicología. Técnologías utilizadas: El sistema fue realizado con Python, Visual Studio Code y se utilizó una base de datos MySQL. Objetivo principal: El objetivo era ayudar a las personas con ansiedad, depresión y miedo en los momentos de crisis. Resultado obtenido: Fue una gran ayuda para las personas que lo utilizaron ya que les dio consejos y recomendaciones para mejorar su estado emocional.

Interfaz gráfica de clientes de apartamentos

Descripción: El proyecto fue la creación de una interfaz gráfica donde los clientes pudieran ver los apartamentos disponibles, los costos, imagenes y así poder contactar al administrador para obtener más información. Técnologías utilizadas: El sistema fue realizado con Python y Neatbeans IDE. Objetivo principal: El objetivo era ayudar a los clientes a visualizar los apartamentos disponibles y a los administradores a tener un mejor control de los mismos. Resultado obtenido: A los clientes les gusto la interfaz gráfica y a la empresa le ayudó a darse a conocer con los apartamentos disponibles.

Implementación de video juegos

Descripción: Este proyecto fue la creación de diferentes video juegos para niños de manera en que fueran entretenidos y educativos. Técnologías utilizadas: El sistema fue realizado con Java y se crearon diferentes juegos como el de la serpiente, Ahorcado, Tic Tac Toe, entre otros. Objetivo principal: El objetivo era ayudar a los niños a aprender de una manera entretenida y educativa. Resultado obtenido: A los niños les gustaron los juegos y les ayudaba a tener pasiencia y concentración.

Sistema de tablero estudiantil y administrativo

Descripción: Este proyecto es un sistema de tablero estudiantil y administrativo en donde los alumnos podían verificar las tareas asignadas por el profesor y los administradores podian verificar las tareas asignadas por los profesores, subir archivos y los profesores podían recibir notificaciones en tiempo real sobre las actuaizaciones, poseía un ingreso con contraseña y doble autenticación. Técnologías utilizadas: El sistema fue realizado con Django y Python por medio de React y se utilizó una base de datos MySQL. Objetivo principal: El objetivo era ayudar a los alumnos a visualizar las tareas de una forma dinámica y a los profesores con el control de las tareas. Resultado obtenido: A los alumnos les gusto el tablero dinámico y a los profesores les ayudó con el control de las tareas.
